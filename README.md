# To‑Do App (Python CLI)

A simple and clean command‑line To‑Do application built in Python.  
This project lets you add, list, and remove tasks, with all data saved in a JSON file.

## Features
- Add new tasks
- List all tasks
- Remove tasks by number
- Tasks are saved in `tasks.json`
- Simple and beginner‑friendly Python code

## How It Works
The app stores tasks in a JSON file so they remain saved even after closing the program.

### Commands
- `list` → Show all tasks
- `add` → Add a new task
- `remove` → Remove a task by number
- `quit` → Exit the app
